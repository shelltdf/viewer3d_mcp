# 产品设计：gltf-mcp（GLTF 模型观察 + MCP）

## 愿景与范围

在浏览器中加载并交互查看 **glTF / GLB** 三维模型，支持旋转/缩放/平移观察；并通过 **Cursor MCP** 与 **HTTP 桥接**让 AI 能远程触发加载与相机重置（交互模式对齐 mindmap 扩展：stdio MCP → 本地桥 → 已打开页面）。

**范围（MVP）**

- 单页 Web 应用（SPA），默认本地或静态托管运行。
- 通过 **URL** 或 **本地文件选择** 加载模型。
- 基础光照与轨道式相机控制。
- **MCP 工具**：查看器在线状态、`load_gltf_url`、`reset_camera`。

**非目标（MVP）**

- 服务端持久化、用户账号、协作编辑。
- 动画时间轴编辑、材质节点图编辑等专业 DCC 能力。

## 用户与成功标准

| 角色 | 成功标准 |
|------|----------|
| 审阅者 | 能在 1 分钟内打开页面并看到模型，操作直观无卡顿（中等体量模型）。 |
| 开发者 | 构建与运行命令文档化，依赖可自动安装。 |

## 人机交互概述

- 主界面采用 **Maya 风格多 Panel 工作区**：左侧全局工具栏、中间 3D 视口区域、左侧 Outliner（按布局出现）、右侧 Attribute Editor Dock。
- 顶部主菜单 `File/Edit/Create/...`；`File` 菜单内包含 URL 加载、本地多文件加载、重置相机。
- 中心区域支持按 `1/2/3/4` 切换 panel 数量：
  - `1`：仅 3D View
  - `2`：Outliner + 3D View
  - `3`：Outliner + 右侧上 3D View + 右侧下 Graph(2D 场景图)
  - `4`：四个 3D 子视口（不显示 Outliner）
- 右侧 Dock 显示当前选中对象属性（名称、类型、可见性、位移/旋转/缩放、UUID），并随选择与变换实时更新。
- 鼠标导航采用 Maya 约定：`Alt+LMB` 旋转、`Alt+MMB` 平移、`Alt+RMB` 推拉；点击 3D 空白区域会取消选择。

> UI 几何与控件布局的权威说明可辅以 `ui.svg`；若未单独维护 SVG，以本文件与 `02-physical` 行为规格为准。
